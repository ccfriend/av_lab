# av_lab
av实验室，用于av相关资料学习分享，立志于让所有人都能学习、理解、探讨音视频领域。

题目愿景很大，自己不一定能在有限的时间里完成完善，希望后来者可以承继前人的优秀经验，不断总结，让所有致力于音视频领域的同学都能茁壮成长。

这里包括：

- 系统介绍大类知识的专栏文章
- 结合Android系统的分析
- 零散的技术总结
- 开源协议、技术规范
- 参考工具
- 测试资源
